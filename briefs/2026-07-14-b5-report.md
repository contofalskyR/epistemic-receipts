## Anomalies / notes

1. **Edit tool persistence:** During this session the Edit/Write tools reported success but changes weren't written to disk in several cases. Bash heredoc/Python was used as a reliable fallback for all file writes. Root cause unclear (possibly a session state issue).

2. **Branch ancestry:** The b5 branch includes B4 commits in its history (b4 was on the base when b5 was created). This is cosmetic — the B5 work is cleanly additive on top.

3. **X-Frame-Options ALLOWALL:** This is non-standard per RFC but widely recognized by browsers. The authoritative framing control is via CSP `frame-ancestors *` which supersedes X-Frame-Options in all modern browsers. The ALLOWALL value handles legacy browsers without CSP support.

4. **No curl matrix from running dev server:** The brief requires not running `next build` on this VPS. The middleware logic was verified by code inspection rather than live curl.

5. **TS errors (pre-existing, not from B5):**
   - `validator.ts`: missing `app/api/search/semantic/route.js` type declarations
   - `app/api/proxy/reader/route.ts`: missing `sanitize-html` module

Both errors existed before B5 and are unrelated to these changes.
